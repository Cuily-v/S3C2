This folder contains Matlab code related to the following paper:
PBLR: an accurate single-cell RNA-seq data imputation tool

The folder was last updated by Lihua Zhang on July 19, 2018.

Our Matlab (>= 2017) code admits a few experimental usages and has not been extensively tested. Please email zsh@amss.ac.cn/ zhanglihua@amss.ac.cn to report bugs.
